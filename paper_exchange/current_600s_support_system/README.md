# 绞吸挖泥船支撑系统 600 s 工况数据

> **历史数据提示：**本目录的 18 个工况使用修复前不完整的桩土 X 运动映射，只保留作运行记录，不再作为桩土接触点响应的物理证据。修复后的单一 RAC 工况见 [`paper_evidence/600s_support_system_contact_kinematics_repair_rac_v19_2026-07-19/`](../../paper_evidence/600s_support_system_contact_kinematics_repair_rac_v19_2026-07-19/)。

本目录是供论文导师和写作者直接使用的简化入口，只包含工况设置、工程响应和运行状态，不包含论文观点、趋势判定或参数推荐。

## 建议阅读顺序

1. `case_table.csv`：先确认18个工况分别改变了什么。
2. `response_data.csv`：查看各工况的工程响应统计量。
3. `soil_midpoint_data.csv`：查看入土深度和土体水平抗力系数两个参数轴上的10个取值点。
4. `run_status.json`：核对计算完成状态和论文使用边界。

## 共同计算条件

- 每个工况计算600 s，时间步长0.1 s，共6001帧、6000个固定Newmark步。
- 全部18个工况均由主校正器完成，未使用救援求解，拒绝步和不收敛步均为0。
- 全部工况启用横摇自由度。
- 环境载荷基础条件：水深35 m，JONSWAP谱有效波高2.0 m，峰值频率0.143 Hz，谱峰升高因子3.3，风速13.8 m/s，均匀流速1.0 m/s。
- 载荷组合通过AQWA环境载荷是否启用以及绞刀载荷采用恒定均值或波动冲击时程进行区分。

## 研究变量

- 缓冲系统变量为蓄能器平均初始压力85、135和185 bar；每条等效支路的蓄能器数固定为9。
- 定位桩实际入土深度为3.5、3.75、4.0、4.25和4.5 m。
- 土体水平抗力系数为2.00、2.625、3.25、3.875和4.50 MN/m^4。
- 3.75 m、4.25 m、2.625 MN/m^4和3.875 MN/m^4为补充的参数中点。

## 响应数据口径

`response_data.csv`中的统计量取缓冲机构行程范围内且时间为60～600 s的样本，采用以下通用工程指标：

- 横摇角和横摇角速度；
- 船体与定位桩相对纵摇角；
- 两条缓冲支路的活塞位移、蓄能器压力和钢丝绳张力；
- 定位桩相对固定土体参考的纵向位移和转角；
- 桩土反力和反力矩；
- 支撑X向力与纵摇力矩的运行时增量。

桩土X位移不是定位桩相对缓冲台车的位移。支撑增量不含预载，物理总支撑量包含预载。

## 使用边界

- 达到缓冲行程边界后的刚性止挡只用于继续完成数值计算，不代表真实液压冲击、结构强度或安全性。
- 桥架铰接点纯反力矩为模型规定的零值。
- 当前18个工况均为单次数值计算，没有重复样本或现场试验标定。
- 当前运行记录仍为 `report_use.allowed=false`，在独立阶段审核通过前，不得写成普适规律、因果结论、最优参数、安全裕度或设计级建议。

完整的逐案指标、哈希和运行封印保存在审计归档目录中；本目录不重复展示这些内部文件。
