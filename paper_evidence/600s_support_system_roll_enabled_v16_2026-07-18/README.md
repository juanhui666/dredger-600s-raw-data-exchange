# 论文导师用 600 s 支撑系统证据包（横摇自由度启用版）

本目录用于支持论文《环境载荷与绞刀载荷作用下绞吸挖泥船缓冲台车及定位桩—土支撑系统动力响应特性》的框架设计、论证审查和图表规划。

这是当前论文的正式数据入口。此前 `600s_support_system_2026-07-18` 目录对应横摇自由度锁定的旧结果，文件保留用于追溯，但已被本目录取代，不得继续用于当前论文中的横摇响应或支撑耦合论述。

## 导师阅读顺序

1. `chatgpt_mentor_prompt_full_outline.md`：论文主线、术语、工程边界和导师任务。
2. `evidence.md`：14 个工况的证据边界、逐案域信息和限定性观察。
3. `case_summary.csv`：逐案响应指标，包含横摇自由响应指标。
4. `relative_to_RAC.csv`：各工况相对共同作用基准工况的描述性变化。
5. `soil_3x3_descriptive.csv`：实际入土深度与土体水平抗力系数 3×3 工况的描述性结果。
6. `run_matrix.json`：14 个工况的变量和分组定义。
7. `matrix_summary.json`：机器可读的完整汇总证据。
8. `post_run_result_seal.json`、`post_review_seal.json`、`ready_seal.json`：运行与终审封印。

## 工况与计算状态

- 14/14 个工况均完成 600 s 计算，每案 6001 帧、6000 个固定 Newmark 步。
- 全部 84000 个时间步均由主校正器收敛，rescue、reject、nonconvergence 均为 0。
- 横摇自由度在全部工况和全部帧中均为 `free_dynamic`，角度和角速度单位分别为 deg、deg/s。
- 载荷组合为 R0、RA、RC、RAC；内部代号仅用于工况表、图例和方法说明。
- 缓冲系统变量为蓄能器初始压力 85、135、185 bar，每支路蓄能器数固定 N=9。
- 定位桩实际入土深度 L=3.5、4.0、4.5 m；土体水平抗力系数 m=2.00、3.25、4.50 MN/m^4。

## 解释边界

- 只允许形成本文 14 个工况、600 s 计算范围内的描述性计算结论。
- 论文主体优先使用行程内响应。行程到限后的刚性止挡只是低频续算代理，不代表真实液压冲击、结构强度或安全性。
- 横摇可以作为自由动态响应讨论，但当前 3×3 工况中并非所有行列都单调，不能写成“支撑增强必然减小横摇”。
- 桥架铰接点纯力矩为模型规定的零值，不能解释为计算发现。
- 支撑总量包含预载；工况比较使用明确的运行时增量口径。
- 不得声称统计显著性、普适因果、实船验证、现场标定、最优参数、设计级适用或安全裕度。

本包不含原始 600 s 压缩时程、求解器代码和内部运行目录。导师若建议引用具体数值或制作新图，必须先由工程审核者确认字段、单位、响应域和证据口径。
